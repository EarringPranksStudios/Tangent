## Features

### Editing

- Inverse Kinematic Bones
- Forward Kinematic Bones
- Sprite Layering
- Sprite Scaling
- Sprite Rotation
- Sprite Offsetting

Keyframes with **29** easing options!

- JSON
- GIF
- seperate PNGs
